Dynamic, High Precision SVG Star Ratings

This SVG uses clip paths to create a star rating. Clip paths allow you to dynamically set the rating using high precision. You are not limited to whole number ratings like 1, 2, 3, 4 or 5. You can use decimal ratings like 4.6 or 3.14159. You simply change the width of the rating__value element (as a percentage) to:

element width = (rating x 20)%

So if you want to portray a rating of 3.7, you set a width of 3.7 x 20 = 74%;

A variation on this would be to set the width of the element in CSS or calculate it using SASS perhaps (although that isn't too much help if you are dynamically generating your star ratings in your HTML from an API call or backend system).

With a bit of SVG wrangling you could easily swap out the stars for any other shape you want (hearts, circles, etc). Just gotta swap in a different element and change up the viewport.

Also you could easily adapt this to a different number of stars, like 4 or 10 or whatever you want really.

This may or may not be the best approach, but it allows some nice precise ratings that can easily be dynamically generated by your backend or templating system.
